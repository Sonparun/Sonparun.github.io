LECTURE - 04 STRING

MULTIPLE INPUT WITH MAP FUNCTION

ฟังก์ชัน map() เป็นฟังก์ชันในตัวใน Python ซึ่งใช้ฟังก์ชันที่กำหนดกับแต่ละรายการของ iterable (เช่น list, tuple เป็นต้น) และส่งคืนรายการผลลัพธ์หรืออ็อบเจ็กต์แผนที่

a,b,c=map(int,input().split())
print(a,b,c)
a,b,c=input().split()
print(a,b,c)

a,b,c=map(float,input().split())
print(a,b,c)

STRING ARE IMMUTABLE

ใน Python stringsจะไม่เปลี่ยนรูป ซึ่งหมายความว่าเมื่อสร้างแล้ว ไม่สามารถเปลี่ยนแปลงได้ การดำเนินการบางอย่าง เช่น การต่อกัน การแก้ไขstrings

friend='Kill'
friend[0]='Q'

กำหนด คำว่า 'Kill' ใน friend
เมื่อเราพยายามเปลี่ยนตัวแรกใน friend เป็น 'W'
จะเห็นว่าไม่สามารถเปลี่ยนได้

STRING SLICING

Python เมื่อเราต้องการตัดเอาข้อความเฉพาะบางส่วนจากตัวแปรที่เก็บข้อมูลประเภท String เราสามารถทำได้โดยการใช้ slice syntax ซึ่งมีรูปแบบการใช้งานดังนี้

ss="Sammy Shark!"
print(ss[4])
print(ss[6:11])
print(ss[:5])
print(ss[7:])
print(ss[-4:-1])
print(ss[6:11])
print(ss[6:11:1])
print(ss[0:12:2])
print(ss[0:12:4])
print(ss[::4])
print(ss[::-1])
print(ss[::-2])

STRING FUNCTION

Len Python คือ คำสั่งสำหรับนับจำนวน กรณีถ้าใช้คำสั่ง len กับข้อความจะเป็นการนับจำนวนตัวอักษร และถ้าใช้คำสั่ง len กับตัวแปร List จะเป็นการนับจำนวนข้อมูลในตัวแปร List สามารถเขียนโปรแกรมได้ดังนี้

fruit='apple'
print(len(fruit))
city="Boston"
index=0
while index < len(city):
    print(city[index])
    index+=1
